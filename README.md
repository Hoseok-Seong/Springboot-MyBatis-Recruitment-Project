## 기술스택

- JDK 11
- Springboot 2.7.8
- MyBatis
- 테스트 H2 DB
- 프로덕션 MySQL DB
- JSP
- redis

## 기능정리

### 1단계 

- 회원가입, 로그인, 메인페이지, 이력서 등록, 수정, 삭제, 목록 보기, 상세 보기, 채용 공고 등록, 수정, 삭제, 목록 보기, 상세 보기

### 2단계 

- 검색, 로그인 기억하기, 아이디, 비밀번호 중복검사, Sha256, Salt 암호화, 지원하기, 지원 취소, 합격, 불합격 기능 

### 3단계 (진행중)

- D-day, 카테고리 검색, 유저 마이페이지 지원 현황, 기업 추천 서비스, 기업 마이 페이지 지원자 현황, 인재 추천 서비스, 북마크
